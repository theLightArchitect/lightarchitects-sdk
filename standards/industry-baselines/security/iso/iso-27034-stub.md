## Publicly-known scope

ISO/IEC 27034 defines a full **application security lifecycle** integrated into application development:

| Lifecycle phase | Coverage |
|-----------------|----------|
| **Inception** | Security requirements identification |
| **Architecture & Design** | Security controls specification |
| **Implementation** | Secure coding practices |
| **Verification** | Security testing + validation |
| **Deployment** | Secure release procedures |
| **Operations** | Monitoring + incident response |

LASDLC v2.4.1 §6.5 `iso_27001_27034_control_checklist.iso_27034_application_security_lifecycle` ships the 6-phase mapping with each phase mapped to LASDLC sections (e.g., Inception → §0.4 + §0.5 + §6; Verification → §7.5 + §7.7 D6).
