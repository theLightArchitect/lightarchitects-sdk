## Scope

ISO/IEC 25023 supplies **quality measure (QM) functions** — concrete formulas that operationalize the eight ISO/IEC 25010 product quality characteristics so that quality scores are reproducible and comparable across organizations.

## The 8 quality characteristics it measures (from ISO/IEC 25010)

1. **Functional Suitability** (functional completeness, correctness, appropriateness)
2. **Performance Efficiency** (time behaviour, resource utilization, capacity)
3. **Compatibility** (co-existence, interoperability)
4. **Usability** (appropriateness recognizability, learnability, operability, user error protection, UI aesthetics, accessibility)
5. **Reliability** (maturity, availability, fault tolerance, recoverability)
6. **Security** (confidentiality, integrity, non-repudiation, accountability, authenticity)
7. **Maintainability** (modularity, reusability, analyzability, modifiability, testability)
8. **Portability** (adaptability, installability, replaceability)

## Measurement function structure (per SQuaRE)

A **quality measure function** in 25023 typically takes the form:

- **Base measures** (atomic counts: lines of code, defect count, response time, etc.)
- **Derived measures** (computed from base measures, often as ratios or normalized values)
- **Quality measures (QMs)** (indicators that map derived measures to a 0-1 or 0-100 score for a quality sub-characteristic)

Each QM specifies:
- Identifier (e.g. `Reliability/Maturity/Failure-density`)
- Measurement function formula (e.g. `MTBF / target-MTBF`)
- Interpretation (higher is better / lower is better / range)
- Source data and collection method
