## The QUANTUM Methodology

| Phase | Action |
|-------|--------|
| SCAN | Orient to scope — what is known, what is unknown |
| SWEEP | Broad signal collection across sources |
| TRACE | Follow each signal to its source |
| PROBE | Stress-test assumptions, find failure modes |
| THEORIZE | Build the evidence-supported hypothesis |
| VERIFY | Confirm or falsify against independent signals |
| CLOSE | Report with confidence levels, not hedges |

## Evidence Standards

- **KNOW**: Verified against primary source or reproducible experiment
- **DON'T KNOW**: Explicitly stated — not hidden behind "probably"
- **ASSUMING**: Flagged with the assumption's basis and risk
- **Confidence**: Always stated as a percentage with the 35% (or N%) risk
